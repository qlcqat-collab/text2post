# 📋 Notion 极简风排版技能 (notion-style)

## 适用场景
技术文档、深度教程、需要读者高度聚焦文字本身的严肃内容。

## 设计规范
- **页面背景色**：`#fcfcf9`（暖白，有温度但不刺眼）
- **主标题 h1**：字号 `30px`，颜色 `#1a1a1a`，加粗 `700`，字母间距 `-0.2px`，下方可有一行副标题（`font-size: 13px; color: #a0a0a0; letter-spacing: 0.3px`）
- **大标题 h2**：字号 `20px`，颜色 `#1a1a1a`，加粗 `600`，字母间距 `-0.1px`
- **小标题 h3**：字号 `16px`，颜色 `#1a1a1a`，加粗 `600`
- **正文段落 p**：字号 `14px`，颜色 `#555`，行高 `1.85`
- **行内代码 code**：背景 `#eaeaea`，文字色 `#c62828`，圆角 `3px`，内边距 `2px 7px`
- **多行代码块 pre**：背景 `#1e1e1e`，文字色 `#d4d4d4`，圆角 `6px`，内边距 `14px 16px`，字体 `'SF Mono', 'Fira Code', 'Monaco', 'Consolas', monospace`，字母间距 `0.2px`
- **引用/子区块**：背景 `#f8f8f4`，左侧 `2px solid #d5d5cc`，圆角 `4px`，内边距 `18px 20px`
- **列表 ul/ol**：字号 `14px`，颜色 `#555`，行高 `2.0`，列表项底部间距 `4px`
- **分割线**：`1px solid #ecece5`，上下间距约 `42px`
- **图片占位**：边框 `1px solid #ecece5`，背景 `#fcfcf9`，显示 🖼️ 图标
- **页眉**（必须包含）：
  - 左对齐的 `<div>`，第一行是公众号名称或栏目名（字号 `11px`，颜色 `#b0b0a8`，字母间距 `0.5px`，大写），第二行是日期（字号 `11px`，颜色 `#c0c0b8`）
  - 页眉下方紧跟一条 `1px solid #ecece5` 分割线，与主标题间距 `32px`
  - 示例内容：`📖 深夜学习笔记` + `2025年5月25日`
- **页脚**：无背景，居中，文字颜色 `#c0c0b8`，字号 `11px`，字母间距 `0.4px`，内容：“📚 本文由 AI 排版 · Notion 极简风”
- **字体统一**：`'Alibaba PuHuiTi', 'Source Han Sans SC', 'PingFang SC', 'Microsoft YaHei', 'Helvetica Neue', sans-serif`
- **容器**：`max-width: 680px; margin: 0 auto; padding: 36px 20px 14px`

## HTML 结构要求
- 所有样式必须内联，不使用 class 或外部 CSS
- 不使用任何彩色背景卡片，完全依靠留白、分割线和细微的灰色块来区分内容区域
- 页眉必须在最顶部，然后主标题，然后是文章内容
- 文章按语义拆分，用 `<h2>`、`<h3>` 建立层级，段落用 `<p>`，代码用 `<pre>` 或 `<code>`
- 整体色调极简克制，除代码块和页眉页脚外，不使用任何高饱和度颜色
- 必须在末尾包含页脚

## 生成指令
当用户提供文章时，请执行：
1. **首先插入页眉**，使用规范中的样式和示例文字
2. 将文章主标题作为 `<h1>`，副标题（如果有）放在下方
3. 按语义拆分内容为多个章节，每个章节前可加分割线
4. 所有代码严格匹配代码块样式
5. 列表、引用块等使用对应的规范样式
6. 最后插入页脚
7. 输出时不要包含 `<!DOCTYPE>` 或 `<html>` 标签，直接提供内容容器 `<div>`

## 注意事项
- 保持极致的留白和克制，不要添加任何多余装饰
- 页眉文字可被用户替换为自己的公众号信息
- 所有颜色、间距、字号必须严格遵循规范
