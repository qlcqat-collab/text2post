# 🍮 羊皮纸阅读风排版技能 (soft-reading)

## 适用场景
深度长文、散文、笔记、需要仿古书卷气息的内容，像在羊皮纸上阅读。

## 设计规范
- **页面背景色**：`#f3ecd8`（陈年羊皮色），并加 `inset box-shadow: 0 0 40px rgba(139,119,80,0.12)` 模拟边缘陈旧
- **主标题 h1**：字号 `28px`，颜色 `#2b2418`，加粗 `700`，字母间距 `0.5px`，可带副标题（字号 `13px`，颜色 `#8b7d65`）
- **大标题 h2**：字号 `20px`，颜色 `#2b2418`，加粗 `600`，字母间距 `0.4px`，可选斜体
- **小标题 h3**：字号 `16px`，颜色 `#2b2418`，加粗 `600`
- **正文段落 p**：字号 `15px`，颜色 `#2e2a22`，行高 `1.9`，段落两端对齐 `text-align: justify`
- **行内代码 code**：背景 `#e8e0ce`，文字色 `#5a3e2b`，圆角 `3px`，内边距 `2px 7px`
- **多行代码块 pre**：背景 `#1f1b16`，文字色 `#c8bfaa`，圆角 `4px`，内边距 `14px 16px`，边框 `1px solid #cfc2a8`，等宽字体
- **引用/子区块**：背景 `#f0e9d5`，左侧 `4px solid #6b5544`，可加轻微内阴影 `inset 0 0 8px rgba(0,0,0,0.03)`
- **列表 ul/ol**：字号 `15px`，颜色 `#2e2a22`，行高 `2.1`，项间距 `4px`
- **分割线**：`2px dotted #c4b998` 作为主要分割；`1px solid #c4b998` 作为次要分割或双线效果
- **图片占位**：虚线边框 `1px dashed #c4b998`，背景 `#f0e9d5`
- **页眉**（必须包含）：
  - 左对齐的栏目名（可加装饰符号如 ❧），字号 `11px`，颜色 `#9a8b72`，字母间距 `0.8px`
  - 右侧页码或编号，字号 `11px`，颜色 `#b8aa91`
  - 页眉下方使用双线：`border-top: 1px solid #c4b998; border-bottom: 1px solid #c4b998; padding: 4px 0;`
- **页脚**：顶部 `1px solid #d6ccb4`，居中，文字颜色 `#b8aa91`，字号 `11px`，内容：“📚 本文由 AI 排版 · 羊皮纸阅读风”
- **字体**：`'Alibaba PuHuiTi', 'Source Han Sans SC', 'PingFang SC', 'Microsoft YaHei', 'Helvetica Neue', sans-serif`
- **容器**：`max-width: 680px; margin: 0 auto; padding: 42px 24px 18px`

## HTML 结构要求
- 所有样式内联，不可用外部 CSS
- 整个页面像一张羊皮纸：背景色 + 内阴影，不使用纯白
- 页眉必须在最顶部，带有仿古双线
- 章节间用粗点状线或实线分割
- 文字颜色和背景保持低对比度暖色调，不刺眼
- 代码块背景最深，但仍需带暖意
- 最后必须包含页脚

## 生成指令
1. 首先生成页眉（包括装饰符号和双线）
2. 放置主标题和副标题
3. 按文章内容拆分章节，章节间插入对应的分割线
4. 所有文本两端对齐，代码使用规范中的 pre 样式
5. 子区块用浅羊皮纸底色 + 深色左边框
6. 末尾添加页脚
7. 输出 HTML 代码片段（无需 <!DOCTYPE> 或 <html> 标签）
