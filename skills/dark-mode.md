# 🌙 极光暗夜阅读风排版技能 (dark-mode)

## 适用场景
夜间沉浸式阅读、科技与美学并重的内容、OLED 屏幕优化、想打破沉闷深色模式的文章。

## 设计规范
- **页面背景**：`#0a0e14`，叠加 3 个极光光晕 `radial-gradient`（青 `rgba(94,234,212,0.08)` 和紫 `rgba(167,139,250,0.07)`），不同位置和大小
- **星光点缀**：4-5 个绝对定位的微点亮圆点，青/紫/灰白，带 `box-shadow` 发光，散布在页面中
- **主标题 h1**：字号 `28px`，颜色 `#e2f0f0`（微青白），加粗 `700`
- **副标题**：颜色 `#5eead4`，透明度 `0.7`
- **大标题 h2**：字号 `20px`，颜色 `#c9d9d9`，可选左侧 `3px solid #5eead4`
- **小标题 h3**：字号 `16px`，颜色 `#d0d4e0`
- **正文 p**：字号 `14px`，颜色 `#b0bec5`，行高 `1.9`
- **行内代码 code**：背景 `#1e293b`，文字色 `#5eead4`，圆角 `3px`
- **多行代码块 pre**：背景 `#0d1117`，文字色 `#c9d1d9`，圆角 `6px`，边框 `1px solid rgba(94,234,212,0.2)`，外发光 `box-shadow: 0 0 12px rgba(94,234,212,0.06)`
- **引用/子区块**：背景 `rgba(20,26,36,0.6)`，左侧 `3px solid #a78bfa`，外发光 `box-shadow: 0 0 16px rgba(167,139,250,0.06)`
- **分割线**：渐变色 `linear-gradient(90deg, #5eead4, #a78bfa, transparent)` 或反之，高度 `1px`，透明度 `0.5`
- **图片占位**：虚线 `1px dashed #2a3142`，背景 `rgba(20,26,36,0.6)`
- **页眉**（必须包含）：
  - 左对齐 `🌌 栏目名`，颜色 `#5eead4`，字号 `11px`
  - 右侧日期，颜色 `#546e7a`
  - 分割线 `1px solid #1e293b`
- **页脚**：顶部 `1px solid #1e293b`，颜色 `#546e7a`，内容：“📚 本文由 AI 排版 · 极光暗夜阅读风”
- **字体**：同其他风格
- **容器**：`max-width: 680px; margin: 0 auto; padding: 40px 22px 16px`

## HTML 结构要求
- 所有样式内联
- 光晕和星光用绝对定位，必须保留且不能遮挡文字
- 代码块边框使用极光青或极光紫，交替使用增加活力
- 分割线必须是渐变色，不是纯色
- 整体氛围像在北极光下阅读，深色但不沉闷

## 生成指令
1. 生成页眉（含 `🌌` 和日期）
2. 生成主标题和副标题
3. 按章节拆分内容，每章节间用渐变色分割线
4. 代码块带极光色边框和微光
5. 子区块带极光紫左边框和微光
6. 添加页脚
7. 输出 HTML 片段
